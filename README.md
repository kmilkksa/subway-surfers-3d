# 🏃 Subway Surfers 3D Clone

Endless runner game built with Three.js - inspired by Subway Surfers.

## 🎮 Features

- **3 Lanes** - Switch between left, center, and right lanes
- **Multiple Obstacles**:
  - 🔴 Low barriers - Jump over them
  - 🔵 High barriers - Slide under them  
  - 🟥 Full blocks - Change lanes to avoid
- **Collectibles** - Coins scattered throughout the track
- **Power-ups**:
  - 🧲 Magnet - Attracts nearby coins
  - 🛡️ Shield - Protects from one hit
  - 🚀 Jetpack - Fly above obstacles
  - ✨ Multiplier - 2x score boost
  - ⚡ Speed Boost - Move faster
- **Progressive Difficulty** - Speed increases as you level up
- **Mobile Support** - Touch controls for mobile devices
- **Polished Graphics** - Shadows, lighting, city buildings

## 🕹️ Controls

| Action | Keyboard | Mobile |
|--------|----------|--------|
| Move Left | ← or A | Left button |
| Move Right | → or D | Right button |
| Jump | ↑ or W or Space | Up button |
| Slide | ↓ or S | Down button |

## 🚀 How to Play

1. Open `game.html` in any modern browser
2. Click **START** to begin
3. Dodge obstacles, collect coins and power-ups
4. Survive as long as possible!
